# Darul Web Agency 🌐

Darul is a modern web agency website built using Vite.js. This project focuses on delivering a clean, responsive, and professional platform to showcase web services and digital solutions.

## ✨ Features

- Responsive design for all devices
- Clean and modern UI
- Smooth user experience
- Optimized performance with production build
- Well-structured React components

## 🎨 Design

The website uses a harmonious blend of blue tones to create a professional, calm, and visually appealing appearance that reflects trust and innovation.

## 🚀 Tech Stack

- React.js
- JavaScript (ES6+)
- HTML5 & CSS3

## 📦 Installation

Clone the repository:

```bash
git clone https://github.com/muhaqib/darulweb.git

Run `npm i` to install the dependencies.

Run `npm run dev` to start the development server.

```
